### Descrição do Projeto
Este projeto foi realizado durante o Programa ONE, da Oracle com a Alura. A proposta da realização deste projeto é praticar a persistência de dados.

Dentro deste programa, os dados referente a livros são coletados de uma API chamada Gutendex. Esses dados são persistidos dentro de um banco de dados (aqui, utilizamos o Postgres).

A partir de uma interação com o usuário, esses dados são coletados do banco de dados, utilizando o JPA (e o JPQL).
